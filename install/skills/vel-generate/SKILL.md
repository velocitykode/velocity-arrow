# Vel Generate

Generate code using the Velocity CLI (`vel`) conventions.

## Commands

- `vel new <name>` - Scaffold a new project
- `vel make:provider <name>` - Create a service provider
- `vel make:migration <name>` - Create a database migration
- `vel make:model <name>` - Create a model
- `vel migrate` - Run migrations
- `vel routes` - List registered routes
