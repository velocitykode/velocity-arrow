# Velwatch Instrumentation

Instrument your Velocity application with Velwatch for observability.

## What it does

- gRPC-based telemetry collection
- Middleware for request tracing
- Event listener for application events
- Context propagation for distributed tracing

## Usage

Register the Velwatch provider and middleware in your application bootstrap.
